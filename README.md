# 爆速受付チャレンジ｜tsubox MINI GAME

有限会社ツーボックスの公式マスコット「ツボック」と「ツボリン」が登場するブラウザミニゲームです。

イベント受付にやってくる来場者を、持っているアイコン（事前登録・当日券・関係者）に合わせて正しいレーンへ誘導します。制限時間は60秒。スマートフォンのタップ、スワイプ、PCの矢印キーで遊べます。

## 遊び方

1. 来場者が持っているアイコンをチェック
2. 同じアイコンの受付レーンをタップして誘導
3. 60秒でできるだけ多くの来場者をさばく（ミスは残り時間 −3秒）

## 技術構成

- 単一HTMLファイル（`index.html`）のみで動作。外部ライブラリ・ビルド不要
- キャラクターはすべてインラインSVGで描画
- 効果音はWeb Audio APIで生成（音声ファイルなし）
- ベストスコアは端末のlocalStorageに保存

## 動かし方

`index.html` をブラウザで開くだけで動作します。
